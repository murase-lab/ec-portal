# 競合分析ツール ユーザーマニュアル

楽天市場の検索結果で、自社商品が競合と比べてどれくらいクリックされるかをAIが予測し、改善提案をしてくれるツールです。

---

## 事前準備

### 必要なサーバー
2つのサーバーを起動する必要があります。

**1. バックエンド（lp-heatmap）**
```bash
cd lp-heatmap
python app.py
# → http://localhost:8085 で起動
```

**2. フロントエンド（ec-portal）**
```bash
cd ec-portal
npm run dev
# → http://localhost:5176 で起動
```

### APIキー設定
`lp-heatmap/.env` に以下を設定：

| キー | 用途 | 必須 |
|------|------|------|
| `RAKUTEN_APP_ID` | 楽天商品検索API Application ID | 必須 |
| `RAKUTEN_ACCESS_KEY` | 楽天商品検索API Access Key | 必須 |
| `GEMINI_API_KEY` | Gemini AI分析（Flash/Pro） | AI分析に必須 |
| `ANTHROPIC_API_KEY` | Claude AI分析（Sonnet） | Claude使用時のみ |

### 楽天API IP制限（重要）
楽天APIはリクエスト元のグローバルIPアドレスで利用制限をかけています。
楽天開発者コンソール（https://webservice.rakuten.co.jp/）の「Allowed IP addresses」に、サーバーを動かすPCのグローバルIPを登録してください。

**以下の場合、楽天検索が動かなくなります：**
- 自宅WiFi → 会社WiFi など、ネット回線を変えた場合
- ルーター再起動でプロバイダからIPが変わった場合
- スマホテザリングに切り替えた場合

**対処法：**
1. ブラウザで「my ip」と検索して現在のグローバルIPを確認
2. https://webservice.rakuten.co.jp/ にログイン
3. アプリの「Allowed IP addresses」に新しいIPを追加（複数登録可能）

## 使い方

ブラウザで `http://localhost:5176/competitor-analysis` を開きます。
サイドバーの「競合分析」からもアクセスできます。

### Step 1: 自社商品情報を入力

2つの入力方法があります。

#### 方法A: 手動入力
1. 「手動入力」タブを選択
2. サムネイル画像をアップロード（JPEG/PNG/WebP）
3. 商品名を入力
4. 価格（税込）を入力

#### 方法B: 楽天URLから取得
1. 「楽天URLから取得」タブを選択
2. 楽天の商品ページURL（`https://item.rakuten.co.jp/...`）を貼り付け
3. 「取得」ボタンをクリック → 商品名・価格・画像が自動入力されます
4. 必要に応じて商品名・価格を修正できます
5. 別のサムネイルを使いたい場合はアップロードで上書きできます

### Step 2: ヒートマップ生成

1. 「ヒートマップ生成（無料）」ボタンをクリック
2. 自社サムネイルの視線予測ヒートマップが生成されます
3. サムネイル画像をクリックすると元画像/ヒートマップを切り替えて確認できます

> ヒートマップは赤い部分ほど視線が集まりやすい箇所を示しています。

### Step 3: 競合商品を検索

1. 検索キーワードを入力（カンマ区切りで複数入力可能）
   - 例: `トートバッグ, レディース, 通勤`
2. 取得件数を選択（10件 or 20件）
3. 「楽天で検索」ボタンをクリック
4. 競合商品の一覧が取得されます

### Step 4: 検索結果シミュレーション

競合検索後、スマホの楽天検索結果風のシミュレーションが表示されます。

- **青い枠** + 「あなたの商品」バッジが自社商品です
- 自社商品の表示位置を変更できます:
  - 「ランダム」: 毎回ランダムな位置に表示
  - 「1番目」「2番目」...: 指定した位置に固定表示

> 実際のスマホ画面と同じ2列表示で、競合の中で自社がどう見えるか確認できます。

### Step 5: AI競合分析

1. 使用するAIモデルを選択:

| モデル | コスト目安 | 特徴 |
|--------|-----------|------|
| Gemini 3 Flash | 約3〜5円/回 | 安くて速い。まずはこれで試すのがおすすめ |
| Gemini 3.1 Pro | 約20〜30円/回 | より高精度な分析 |
| Claude Sonnet 4.6 | 約31円/回 | 高品質な分析（APIキー別途必要） |

2. 予測コストが表示されます（競合件数に応じて変動）
3. 「AI競合分析を実行」ボタンをクリック
4. 1〜2分程度でAI分析結果が表示されます

### AI分析の出力内容

- **CTRランキング予測表**: 全商品のスコア一覧（自社は★マーク）
- **総合評価**: 予測順位・スコア・一言評価
- **因子別の詳細分析**:
  - サムネイル競争力（50%配分）: 強み・弱み・ヒートマップ所見
  - 商品名ファーストライン（30%配分）: 語頭30文字の評価・改善案
  - 価格ポジショニング（20%配分）: 価格帯の位置・実質価格比較
- **改善戦略**: サムネイル改善（最優先）→ 商品名改善 → 総合戦略

## サムネABテストからの連携

「サムネABテスト」ページでAI診断を実行した後、「このサムネで競合比較へ」ボタンが表示されます。クリックすると、推奨サムネイルのデータを引き継いで競合分析ページに移動できます。

## よくある問題と対処法

### 「通信エラー: Failed to fetch」
- lp-heatmapサーバー（ポート8085）が起動しているか確認してください
- `cd lp-heatmap && python app.py` で再起動

### 楽天検索で「400エラー」
- `.env` の `RAKUTEN_APP_ID` と `RAKUTEN_ACCESS_KEY` が正しいか確認
- 楽天開発者コンソールでIPアドレスが登録されているか確認

### 楽天検索で「403エラー（CLIENT_IP_NOT_ALLOWED）」
- PCのグローバルIPアドレスが変わった可能性があります
- https://webservice.rakuten.co.jp/ で「Allowed IP addresses」を更新してください

### 楽天URLからの取得で価格が0になる
- 一部のショップでは価格の取得が困難な場合があります
- 手動で正しい価格を入力してください

### AI分析が「タイムアウト」する
- 競合20件の分析は時間がかかります。再度実行してください
- 10件に減らすと速くなります

### ヒートマップ生成でエラー
- lp-heatmapサーバーを再起動してみてください
- OpenCV（`pip install opencv-python`）がインストールされているか確認
